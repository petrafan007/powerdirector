# Vision (PowerDirector)

PowerDirector is a production-focused fork of the OpenClaw project. Objectives:
- Keep upstream OpenClaw parity for gateway, tools/skills, channels, cron, nodes, docs.
- Maintain PowerDirector-specific UX: chat attachments, terminal, provider/model selector, media rendering.
- Stay operationally transparent: clear runtime controls, explicit schemas, auditable configs.
- Ship via GitHub with reproducible builds and no bundled secrets.

Upstream/inherited components (extensions, mobile/desktop apps, some skills) are included for continuity but may be “not supported or still in progress” under PowerDirector; validate before production use.
